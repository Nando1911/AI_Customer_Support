![Logo del proyecto](images/demo.gif) <!-- Opcional: tu GIF de demo como “portada” -->

## Descripción
AI Customer Support es un asistente inteligente capaz de responder preguntas frecuentes de clientes utilizando inteligencia artificial. Este proyecto permite automatizar la atención al cliente, ofreciendo respuestas rápidas y consistentes, y mostrando un ejemplo de cómo integrar IA en aplicaciones empresariales.

---

## Características
- Responde preguntas frecuentes (FAQ) de manera automática.  
- Analiza y procesa datos de soporte para ofrecer respuestas precisas.  
- Interfaz de consola simple para interactuar con el asistente.  
- Generación de reportes visuales (opcional, según el proyecto).  

---

## Tecnologías utilizadas
- Python 3.11+  
- OpenAI API (GPT-4o-mini)  
- Pandas  
- Matplotlib  
- JSON/CSV para almacenamiento de preguntas y respuestas
